[ 오토헌터 프리미엄 v7.0 사용 설명서 ]

안녕하세요! 가장 스마트한 이미지 인식 기반 사냥 보조 프로그램 '오토헌터 v7.0'을 이용해 주셔서 감사합니다.
본 프로그램은 이미지 인식을 통해 캐릭터의 위치를 파악하고, 자동으로 사냥 및 상점 판매를 수행합니다.

-----------------------------------------------------------
1. 시작하기 전에 (필수)
-----------------------------------------------------------
- 이 프로그램은 '관리자 권한'으로 실행하는 것을 강력히 권장합니다.
- 게임 화면은 창 모드로 설정하시고, 배율(DPI)은 100%를 권장합니다.
- Python으로 실행 시: 'pip install -r requirements.txt'를 먼저 수행하세요.
- EXE로 실행 시: 제공된 'AutoHunter_v7.exe'를 바로 실행하시면 됩니다.

-----------------------------------------------------------
2. 주요 단축키
-----------------------------------------------------------
- [ F12 ] : 사냥 시작 / 일시 중지
- [ F11 ] : 원클릭 자동 판매 루틴 즉시 실행 (수동)
- [ F1  ] : (좌표 설정 모드 시) 미니맵 좌상단 지정
- [ F2  ] : (좌표 설정 모드 시) 미니맵 우하단 지정

-----------------------------------------------------------
3. 주요 기능 설정 방법
-----------------------------------------------------------
(1) 미니맵 인식 (가장 중요!)
- '메인' 탭에서 [영역 드래그] 버튼을 누르고 게임의 '미니맵' 영역을 마우스로 드래그하세요.
- 미리보기 창에 캐릭터(녹색 점)가 정확히 표시되어야 사냥이 시작됩니다.

(2) 원클릭 자동 판매 (F11)
- '기능' 탭에서 1~4번 캡처 버튼을 사용해 실제 게임 내 버튼들을 캡처하세요.
  1. 상점 아이콘 (인벤토리 내)
  2. 장비 일괄판매 버튼
  3. 최종 확인 버튼
  4. 상점 나가기 버튼
- 설정 후 F11을 누르면 사냥 중단 -> 상점 이동 -> 판매 -> 사냥 재개가 자동으로 진행됩니다.

(3) UI 배율 설정
- 프로그램이 너무 크거나 작다면 '기능' 탭의 [UI 배율] 메뉴에서 0.5 ~ 1.5배까지 조절 가능합니다.

(4) 펫먹이 자동 사용
- '스킬' 탭에서 [펫먹이 주기(분)]와 사용할 키를 설정하면 시간에 맞춰 자동으로 사용합니다.

-----------------------------------------------------------
4. 문제 해결 (FAQ)
-----------------------------------------------------------
Q: 캐릭터를 못 찾아요!
A: '스킬' 탭의 [인식 여유분] 수치를 조금씩 높여보거나, 미니맵 영역을 다시 드래그해 보세요.

Q: 판매 루틴이 엉뚱한 곳을 클릭해요!
A: '기능' 탭에서 각 버튼 이미지를 더 정밀하게(버튼 테두리 포함) 다시 캡처해 보세요.
