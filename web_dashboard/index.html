10. 문의사항

🛠️ 로그 및 복구 안내

Mac mini 대시보드 접속이 되지 않을 때 아래 순서에 따라 복구를 시도하세요.

1
Tailscale 연결 확인
Galaxy Tab / PC에서 Tailscale VPN이 연결되어 있는지 확인합니다.
2
대시보드 접속 시도
브라우저에서 Mac mini 대시보드 URL에 직접 접속해봅니다.
3
접속 실패 시 SSH 접속
Termius 앱을 열고 Mac mini SSH 호스트에 접속합니다.
4
'서버 상태 확인' Snippet 실행
Termius Snippets에서 '서버 상태 확인'을 실행하여 프로세스 상태를 점검합니다.
5
필요 시 '대시보드 재시작' Snippet 실행
대시보드 서버가 중단된 경우 '대시보드 재시작' Snippet을 실행합니다.
