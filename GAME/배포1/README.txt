[ 오토헌터 프리미엄 v6.0 사용 안내 ]

1. 필수 설치 요소 (Python으로 실행 시)
   - 파이썬이 설치된 환경에서 다음 명령어를 입력하여 필요한 라이브러리를 설치하세요.
     pip install -r requirements.txt

2. 실행 방법
   - 실행 파일(.exe)을 이용하거나, 파이썬에서 auto_hunter_gui_v6_0.py를 실행하세요.

3. 주요 단축키
   - F12: 사냥 시작 / 중지
   - F1: 미니맵 좌상단 지정 (영역 지정 모드 시)
   - F2: 미니맵 우하단 지정 (영역 지정 모드 시)

4. 주요 기능
   - 제자리 사냥 모드: 설정된 범위(1~10px) 내에서 미세하게 움직이며 사냥
   - 거탐 대응: 하얀색 도형 추적 및 관성 추적 기능 포함
   - 주기적 DEL 키: 설정한 분 단위로 자동 입력

5. 주의 사항
   - 미니맵 영역을 정확히 지정해야 캐릭터 위치를 인식합니다.
   - 관리자 권한으로 실행하는 것을 권장합니다.
