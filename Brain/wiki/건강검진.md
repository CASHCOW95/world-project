# 건강검진 (Linting)

지식 베이스가 커질수록 발생하는 노후화, 모순, 누락된 연결을 해결하는 프로세스입니다.

## 주요 점검 항목
1. **정합성(Consistency)**: 서로 다른 페이지에서 상충되는 정보가 있는지 확인.
2. **최신성(Recency)**: AI 업계처럼 변화가 빠른 분야에서 너무 오래된 정보([[AI 트렌드]])가 있는지 체크.
3. **누락된 연결(Missing Links)**: 분명히 관련이 있는데 [[위키링크]]가 걸려 있지 않은 페이지 탐색.
4. **데드 링크(Dead Links)**: 삭제되거나 변경된 페이지 참조 수정.

## 운영 방법
- 클로드 코드에게 "위키 건강검진 한번 하자"라고 요청.
- AI가 제안하는 수정안을 검토 후 일괄 반영.
