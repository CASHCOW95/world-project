# 지식의 복리와 사고 모델

## 💡 쉬운 설명 (Concept)
지식의 복리는 "공부한 게 쌓이고 쌓여서 나중엔 저절로 돈을 벌어다 주는 힘"을 말해요. 경제 뉴스 한 줄, 노래 연습 기록 하나가 따로 놀 때는 힘이 없지만, 이게 서로 연결되면 "어? 노래 연습할 때의 원리를 주식 매매에도 써볼 수 있겠는데?" 같은 기발한 아이디어가 튀어나오게 됩니다.

## 🔍 원문 비교 (Comparison)
| 핵심 포인트 | 쉬운 해설 | 원문 내용 (Reference) |
| :--- | :--- | :--- |
| 상호 참조 | 지식끼리 대화하게 함 | "새로운 자료가 추가될 때마다 기존 자료와 연결... 의외의 연결점이 인사이트가 됨" |
| 사고 모델 | 세상을 보는 틀 | "한 분야의 원리를 다른 분야에 적용... 경제지식과 AI 트렌드가 연결되어 자동화 전략 도출" |
| 복리 발생 | 시간이 갈수록 가치 폭발 | "지식이 임계점을 넘는 순간 AI는 내가 생각지 못한 수익 모델을 제안하기 시작함" |
