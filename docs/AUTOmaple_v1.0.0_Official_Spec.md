# AUTOmaple 정식 버전 v1.0.0 통합 명세서 (Master Original)

## 1. 정식 버전 개요
본 문서는 AUTOmaple 프로젝트의 공식 **v1.0.0 원본** 데이터와 향후 업데이트를 위한 기술적 토대를 정의합니다.

- **버전**: v1.0.0 (Master Original)
- **최종 업데이트**: 2026.06.03
- **실행 파일**: `deploy\dist\AUTOmaple_v1.0.0\AUTOmaple_v1.0.0.exe`
- **백업 상태**: `deploy\backup\v1.0.0\` 폴더에 소스 및 빌드본 영구 보존 중.

## 2. 핵심 기술 사양 (Core Specifications)

### 2.1 Human-Engine (지능형 마우스 컨트롤)
- **알고리즘**: 베지에 곡선(Bézier Curve) 기반 궤적 생성 및 가변 선형 보간(Lerp) 적용.
- **가속도 제어**: 타겟과의 거리에 따른 가속/감속 시뮬레이션으로 탐지 시스템 우회.
- **인간 특성 재현**: 도달 지점 부근의 오버슈트(Overshoot) 및 미세 떨림(Micro-shaking) 효과 탑재.

### 2.2 도형 추적 엔진 (Vision Engine)
- **전처리**: `game_background_pattern.png`를 활용한 정적 배경 차분 알고리즘.
- **인식**: `anti1.png`에서 타겟 색상(HSV)을 동적으로 추출하여 정밀 마스킹 처리.
- **최적화**: 모폴로지 연산 및 종횡비 필터링을 통해 50FPS급 실시간 추적 루프 성능 확보.

### 2.3 시스템 대시보드 및 UI
- **실시간 지표**: `create_data_row` 메서드를 통해 가동 시간, 누적 사냥, 명령 횟수, 탐지 기록을 실시간 렌더링.
- **동적 제어**: F2(엔진 시작), F3(엔진 중지) 단축키 및 UI 상태 실시간 동기화.

## 3. 향후 업데이트 로드맵 (Next Step: v1.0.1)

### 3.1 Lucas-Kanade 광학 흐름(Optical Flow) 도입
- **목표**: 현재의 배경 차분 방식을 넘어, 픽셀의 이동 벡터를 추적하는 **Lucas-Kanade 알고리즘**을 도입하여 정밀도를 극대화함.
- **기대 효과**:
  - 투명 도형이 배경과 겹치거나 매우 빠른 속도로 비선형 이동을 할 때의 추적 탈조 현상 완벽 방지.
  - 특징점(Feature Points) 기반의 정밀한 드래그 상태 유지력 확보.
