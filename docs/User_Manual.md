# AUTOmaple v9.7.0 (2026.06.02) 사용 설명서

1. 실행 및 권한 설정
- 프로그램 실행 시 반드시 '관리자 권한으로 실행'을 선택함. 관리자 권한 미부여 시 전역 단축키가 작동하지 않을 수 있음.
- 창모드 사용 권장, 전체화면 불안정

2. 미니맵 영역 지정
- 메인 화면의 [영역 드래그] 버튼을 클릭함.
- 게임 내 미니맵의 좌측 상단부터 우측 하단까지 마우스로 드래그하여 영역을 선택함.
- 라이브 프리뷰 창에 캐릭터 위치가 정상적으로 표시되는지 확인함.

3. 세부 매크로 설정
- 작동 모드: 이동/제자리 모드를 선택하고 범위 슬라이더를 조절함.
- 스킬/딜레이: 공격 및 이동 딜레이(ms 단위)를 설정함. 
- 단축키 설정: 인게임 키셋팅에 맞춰 공격(ATT), 이동(MOV), 점프(JMP) 키를 할당함.

4. 단축키 운용
- 사냥 시작/중지: F12 키를 누름 (토글 방식). 
- 강제 판매: F11 키를 누름 (이미지 매칭 방식).

5. 설정 저장 및 로드
- 현재의 모든 수치를 저장하려면 상단 [저장] 버튼을 클릭함.
- 콤보박스에서 사냥터 이름을 관리 가능함.

6. 특수 기능 (도형거탐 감지)
- 시스템 환경/스킬 탭에서 '도형거탐 감지' 활성화 시, 화면 내 특정 형태의 거탐을 분석함.
- 거탐 알람: 감지 시 사운드 경고 여부를 결정함.
