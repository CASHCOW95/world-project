- **공식 버전**: **v1.0.0 (Master Original)**
- **릴리즈 일자**: 2026.06.03
- **현재 상태**: 모든 초기 런타임 오류 해결 및 핵심 알고리즘 안정화 완료.

## 2. v1.0.0 핵심 기술 명세 (The Original)

### 2.1 Human-Engine (지능형 마우스 제어)
- **가변 속도 제어**: 타겟 거리($d$)에 따라 LERP 인자를 0.15~0.8 사이로 동적 조절.
- **베지에 궤적**: 자연스러운 곡선형 이동을 구현하여 단순 매크로 감지 시스템 우회.
- **인간 특성 재현**: 도달 지점에서의 미세 진동(Micro-shaking) 및 물리적 관성(Overshoot) 시뮬레이션 적용.

### 2.2 비선형 도형 추적 시스템
- **동적 색상 학습**: `anti1.png` 이미지에서 타겟의 HSV 범위를 실시간으로 추출.
- **배경 제거 로직**: `game_background_pattern.png`를 참조 데이터로 사용하여 움직이는 객체만 선별적으로 추출(`absdiff`).
- **성능**: 루프 정밀도를 0.02s(50FPS)로 고도화하여 당구공처럼 튀는 불규칙 경로에 완벽 대응.

### 2.3 시스템 대시보드 및 안정화
- **실시간 모니터링**: CPU/RAM 점유율, 가동 시간, 누적 명령 횟수 표시.
- **오류 해결**: `create_data_row` 메서드 정식 구현으로 UI 초기화 크래시 완벽 차단.
- **단축키**: F2(시작), F3(중지) 전역 단축키와 UI 상태 실시간 동기화.

## 3. 버전 관리 및 백업 원칙

### 3.1 업데이트 규칙 (0.0.1 Increment Rule)
- 모든 수정 및 기능 업그레이드 시 버전 번호를 **0.0.1** 단위로 상향한다.
- 예: v1.0.0 (원본) → v1.0.1 (Lucas-Kanade 도입) → v1.0.2 ...

### 3.2 정식 원본 백업 (v1.0.0)
- **백업 위치**: `deploy\backup\v1.0.0\`
- **포함 파일**:

  - `dist/`: 즉시 실행 가능한 v1.0.0 정식 배포 패키지.
