# AI 블로그 운영 에이전트 벤치마킹 통합 분석서 (개발자 전달용)

## 프로젝트 방향성

본 프로젝트는 단순한 "AI 블로그 글 생성기"가 아니다.

시장에 판매되고 있는 상위권 자동화 프로그램들을 분석한 결과, 실제 경쟁력은 글쓰기 자체가 아니라 다음 영역에 있었다.

```text
키워드 발굴
→ 클러스터 생성
→ 자료 수집
→ 글 생성
→ 이미지 생성
→ 자동 발행
→ 내부링크 구축
→ 텔레그램 알림
```

사용자는 글을 작성하고 싶은 것이 아니라

"시간을 절약하고 자동으로 블로그를 운영하고 싶어한다."

따라서 제품 포지셔닝은

```text
AI 글쓰기 프로그램
X

AI 블로그 운영 에이전트
O
```

로 정의한다.

# 최종 결론

이 프로젝트의 핵심은 AI 글쓰기가 아니다.

실제 가치 비중은

```text
키워드 선정      30%
자료 수집        20%
클러스터 생성    20%
내부링크 엔진    15%
발행 자동화      10%
글 생성          5%
```

수준으로 판단된다.

따라서 개발 리소스는 프롬프트 최적화보다

```text
황금키워드 엔진
+
클러스터 엔진
+
내부링크 그래프 엔진
+
자동 발행 엔진
```

구축에 우선 투입하는 것이 바람직하다.
