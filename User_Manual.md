# AUTOmaple v8.8 (2026.05.26) 사용 설명서

1. 실행 및 권한 설정
- 프로그램 실행 시 반드시 '관리자 권한으로 실행'을 선택함. 관리자 권한 미부여 시 전역 단축키가 작동하지 않을 수 있음.
- 창모드 사용 권장, 전체화면 불안정

2. 미니맵 영역 지정
- 메인 화면 우측의 [미니맵 영역 드래그 설정] 버튼을 클릭함.
- 게임 내 미니맵의 좌측 상단부터 우측 하단까지 마우스로 드래그하여 영역을 선택함.
- 라이브 프리뷰 창에 캐릭터 위치가 정상적으로 표시되는지 확인함.

3. 세부 매크로 설정
- 작동 모드: 좌우 이동 시 경계 슬라이더를 조절함. 제자리 사냥 시 움직임 폭을 설정함.
- 스킬/정밀도: 공격 및 이동 딜레이(ms 단위)를 설정함. 인식 정밀도는 기본 1.0을 권장함.
- 단축키 설정: 인게임 키셋팅에 맞춰 공격, 이동, 점프 키를 할당함.

4. 단축키 운용
- 사냥 시작: F8 키를 누름. 엔진 가동 시 'AI 오토메이션 엔진이 시작되었습니다' 로그가 기록됨.
- 사냥 중지: F9 키를 누름. 즉시 모든 키 입력을 중단하고 대기 상태로 전환됨.
- 강제 판매: F11 키를 누름 (현재 업데이트 준비 중 안내만 출력됨).

5. 설정 저장 및 로드
- 현재의 모든 수치를 저장하려면 상단 [설정 저장] 버튼을 클릭함.
- 콤보박스에서 사냥터 이름을 입력하거나 선택하여 프로필별로 관리 가능함.

6. 시스템 환경 설정
- 창 항상 맨 위로 고정: 게임 화면 위로 UI를 상시 노출함.
- 거탐 알람: 시스템 환경 탭에서 체크박스를 통해 사운드 경고 여부를 결정함.
